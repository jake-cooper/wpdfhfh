== Description ==

[youtube http://www.youtube.com/watch?v=mPAgVUIFCSs]

[Full documentation](http://siteorigin.com/button-widget/) is available on SiteOrigin.

This plugin gives you a simple button widget that you can add to your widgetized area or [Page Builder](http://wordpress.org/plugins/siteorigin-panels/) pages.

You can choose the color of your button and text using a color picker for virtually unlimited customization. At the moment the widget ships with 3 different styles. These are:

* Atom: A standard web button.
* Wire: Just an outline of a button.
* Flat: A flat, smooth button for modern websites.

In addition, you can choose your padding, font size and button rounding for some really unique looking buttons.
